## Qumulo Frontend App

### Run the project locally

1. Install dependencies

```
npm install
```

2. Copy the example `.env`

```
cp .env.example .env
```

3. Run the project

```
npm run dev
```

### Environment variables

Please change `API_URL` to the server endpoint
